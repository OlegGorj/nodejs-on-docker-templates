nodejs-on-docker-templates
==========================


This repository contains the sources for the following [docker](https://docker.io) base images:
- [`OlegGorJ/nodejs`](/base)
- [`OlegGorJ/nodejs-runtime`](/runtime)
- [`OlegGorJ/nodejs-hello`](/hello)


Lineage
===
